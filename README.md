# EKART (Flipkart-style React E-commerce Demo)

## Features
- Landing page with categories + hero banner
- Category-wise product listing
- Product details page
- Cart with qty +/- , remove, total calculation
- Mock payment page -> Success page
- Header with search + cart badge
- Footer + clean CSS (Flipkart-like)

## Run locally
```bash
npm install
npm start
```
